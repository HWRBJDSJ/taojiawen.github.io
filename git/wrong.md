
# 两种远端本地不同步错误
## 如果遇到需要解决代码冲突的问题

	1.git stash对我们的本地代码进行入栈操作

	2.git pull 将我们远端仓库的代码拉取下来

	3.git stash pop将我们的本地代码进行出栈(远端代码和本地代码有冲突的部分会出现
